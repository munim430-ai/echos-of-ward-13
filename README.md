# Echos of Ward 13 - Horror Game Landing Page

A creepy yet funny landing page for the horror game **Echos of Ward 13**, created by **Humayra Mahjabin**.

## Game Story

You wake on a cold metal table with no memory. A note pinned to your chest reads: *"The Dark hears your heartbeat. Find the three keys before it finds you."*

Ward 13 is waiting. So is **The Patient**—a 7-foot-tall entity with no eyes that hunts by sound alone. Your flashlight is dying. Your fear is rising. And something in the darkness is listening to every breath you take.

## Features

- **Dying Flashlight**: Battery drains in real time. Darkness is not your friend.
- **Fear System**: Stay calm or lose your mind. Literally.
- **Three Keys**: Find them. Hide from The Patient. Escape.
- **The Patient**: 7 feet tall. No eyes. Hunts by sound. Very rude.

## How to Deploy on GitHub Pages

1. Fork or clone this repository
2. Go to Settings → Pages
3. Select "Deploy from a branch"
4. Choose main branch and save
5. Your site will be live at `https://yourusername.github.io/echos-of-ward-13/`
